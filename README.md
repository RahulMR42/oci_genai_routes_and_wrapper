## Sample Routes and wrapper on OCI Generative AI (For demo/non prod usage only)

### Variables 

##### Optional

```
export OCI_REGION = <OCI REGION> #Default is us-chicago-1
export OCI_AUTH_PROFILE = <OCI AUTH Profile> # Default is DEFAULT
export OCI_AUTH_TYPE = <OCI Auth type> # Default is API_KEY,Other option is INSTANCE_PRINCIPAL
export APP_PASSWORD = <Password for API> #If not set a password will be autogenerated
export APP_USER_NAME = <user name>#Default is api_user
export PORT = <app port> # default 5000
```

### Install dependencies

```
poetry lock
poetry install
```

### Run the application 

```
poetry run api #load the application using <ip | 0.0.0.0>:5000/docs
```
